    参考链接：墨者修齐：https://www.yuque.com/mo-college/weekly

接口文档

1 首页

  • 接口说明： 基础接口
  • 接口地址： http://www.51jcjgzy.cn/api/v1/vis/index
  • 接口类型： GET

1.1 请求参数 :

参数名称 类型 出现要求 描述

1.2 返回示例：

&#123;
    "code": 1, //1:成功 0:失败
    "clasdep": &#123;
        "信息工程系": ["19物流信息技术班"]
    &#125;,
      "students": &#123;
      "all": 34, //所有学生超过37.0的统计
        "now": 0   //当天所有学生超过37.0的统计
    &#125;,
      "teachers": &#123;
            "all": 34, //所有教师超过37.0的统计
        "now": 0   //当天所有教师超过37.0的统计
    &#125;
&#125;

2 体温数据统计

  • 接口说明： 体温统计
  • 接口地址： http://www.51jcjgzy.cn/api/v1/vis/bardata
  • 接口类型： GET

2.1 请求参数 :

参数名称 类型 出现要求 描述
type string R 如果查询系typ为1,如果是班级type为2
department string R
clas string 如果查询系不需要此参数

2.2 返回示例：

&#123;
  "code":1,
  "data":[
    &#123;"interval":"35.0-36.0","num":13&#125;,
    &#123;"interval":"36.0-37.0","num":118&#125;
  ]
&#125;

3 武汉数据统计

  • 接口说明：是否去过武汉数据
  • 接口地址： http://www.51jcjgzy.cn/api/v1/vis/wuhandata
  • 接口类型： GET

3.1 请求参数 :

参数名称 类型 出现要求 描述
type string R 如果查询系typ为1,如果是班级type为2
department string R
clas string 如果查询系不需要此参数

3.2 返回示例：

&#123;
  "code":1,
  "data":[
    &#123;"data":"y","num":12&#125;,    //去过的人
    &#123;"data":"n","num":275&#125;    //没过去的人
  ]
&#125;

编写：武杰
