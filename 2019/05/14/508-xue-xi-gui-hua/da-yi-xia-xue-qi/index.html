计算机大一新生下学期学习规划

后端 前端 爬虫 Android
508学习规划
发布日期:   2019-05-14
阅读次数:  

第一个月

  • 学习linux操作系统
  • 推荐文档：
    • Linux
    • ubuntu
  • 推荐软件:
    • vmware
  • 推荐视频:
    • 王道-操作系统
    • linux 提取码：jnt0
    • Linux入门、系统服务、高级管理
  • 建议:
    • 推荐ubuntu系统桌面版
    • 可以在慕课网或者网易云课堂之类的查看免费视频
    • 把自己的或者实验室的电脑修改为ubuntu系统,当然也可以使用vmware安装虚拟机.

第二个月

  • 确认好方向之后开始深入学习,并学习一门数据库
  • 推荐视频
    • 计算机科学CS50-哈佛大学
  • 推荐文档:
    • Android-原生
    • Android-Flutter
    • Web后端-Spring
    • 用来学习 spring boot 的项目
    • Web后端-Node.js
    • Web前端-ES6
    • Web前端-React.js
    • Web前端-Vue.js
    • 爬虫-Python
    • 爬虫-Node.js
  • 建议:
    • 所有的学习建议在ubuntu下
    • 不论是前端或者后端难度其实是差不多的
    • 所有的东西以官方文档为标准,写博客或者视频的不一定是正确的
    • 新手可以使用MySql数据库

第三个月

  • 根据自己的实际情况选择看视频或者看文档
  • 推荐视频:
    • 参考网站中508资源列表
  • 建议
    • 有一个朋友和你一起学习讨论是非常重要的
    • 定期交流学习心得

第四个月

  • 学习git与github
  • 推荐文档:
    • git
    • github
  • 建议
    • git是大部分公司都会使用的代码管理工具,在面试时有github账号是会加分的
    • 学会github会让你搜索资源的能力提升很多
    • 试着使用github寻找一些好的资源并分享

第五个月

  • 总结自己学到东西

  • 推荐

    • 完善新闻客户端加入网页版或者后端
    • 爬取一些简单的网站
    • 一起和朋友使用git合作完成一个项目
    • 程序员应该访问的最佳网站中文版

  • 建议

    • 不要高频率爬虫(爬虫现在很多是犯法的)
    • 多使用github对找工作是有帮助的

其他学长建议

  • 这个时候和大二大三的学长参加一些国家级的比赛是非常有帮助的

一些比赛与博客地址

  • 博客
    • CSDN
    • 掘金
    • V3EX
    • CNode社区
    • 开源中国
    • InfoQ
    • jdk8中文文档
  • 比赛
    • 大学生软件杯
    • 发现杯大奖赛
    • 中国软件杯双创大赛官网
    • 蓝桥杯
    • DC竞赛
