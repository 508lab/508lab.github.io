计算机大二上学期学习规划

后端 前端 Android
508学习规划
发布日期:   2019-05-14
阅读次数:  

学生优惠权益

  • Student-resources

第一个月

  • 这个时候选择专升本或者继续学习技术,复习之前的东西。专科可以直接考研！
  • 推荐文档：
    • 为什么要专升本
  • 推荐平台:
    • 在浏览器使用真实环境学习技术
    • Build your own (insert technology here)
  • 建议:
    • 强烈建议专升本,有机会可以去好的大学看一下
    • 大公司只需要985 211的本科,硕士什么的
    • 当然不论做什么人品和能力才是最重要的

第二个月

  • 总结上一学期的东西,学习一些流行框架

  • 推荐框架:

    • 后端-Spring boot
    • 后端-Egg.js
    • 前端-Ant Design
    • 前端-阿里飞冰
    • 前端-Element
    • 混合开发-React Native
    • 可视化开发-Three.js
    • 混合开发-Taro

  • 建议:

    • 根据自己的情况选择不同的框架

第三个月

  • 可以学一下Go或者Python
  • 推荐视频:
    • Go
    • Python
  • 推荐文档
    • Python在线文档
    • Go在线文档
  • 建议
    • 还在学校的时候可以多学习一些语言或者框架
    • 每个语言或者框架都有他自身的优点与不足

第四个月

  • 试着看一些系统或者框架的源代码
  • 推荐:
    • Android
    • Spring
    • Egg.js
    • Ant design pro
  • 建议
    • 查看源代码对提升自身软件设计能力有很大的帮助
    • 看源码是一个很好的习惯,需要保持

第五个月

  • 多看一些书,比如关于设计模式、面向对象设计之类的
  • 推荐书籍
    • Head First设计模式
    • 大话设计模式
    • 设计模式之禅
    • hacker-laws
    • 成为计算机工程师的完整计算机科学学习计划
  • 建议
    • 可以大概看看,但不要认真.
    • 也可以买一些关于高级语言的书

其他学长建议

  • 最主要还是学校的比赛,这个对找工作专升本都有帮助
